Passo a passo para modificar e mexer no codigo com git:

1: git pull origin main (tome cuidado, caso de algum erro tem que ver se não tem alterações feitas por você que estão conflitando)
2: git status                 # Opcional: Ver o que mudou
3:git add .                  # Ou git add <arquivo_especifico>
4:git commit -m "Sua mensagem descritiva aqui"
5: git push origin main       # Ou o nome da sua branch principal
